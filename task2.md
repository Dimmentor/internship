# Этап 2. Инструменты

Добавьте в систему инструменты для своих агентов.

Предпочтительно использовать FastMCP, но допускается использование LangChain Tools.

Инструмент должен выполнять реальную полезную работу.

Например:

* работа с файлами;
* работа с Git-репозиторием;
* выполнение команд ОС;
* взаимодействие с внешними API (например, получение погоды или курсов валют);
* поиск информации в интернете;
* работа с базой данных;
* генерация файлов;
* любая другая полезная автоматизация.

Использование моков допускается только в том случае, если интеграция с реальным сервисом невозможна в короткие сроки или слишком трудоёмка.

Желательно, чтобы инструмент принимал параметры и возвращал результат, который действительно используется агентами при формировании ответа.
